# Lane-Switch Survivor prototype

Minimal first playable slice for testing if 3-lane dodge survival is fun.

## Run

From this folder:

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.

You can also open `index.html` directly, but a local server is safer for browser consistency.

## Controls

- Left / Right arrow
- A / D
- Tap left or right side of the playfield
- Swipe horizontally on the playfield
- Restart button, or Space after game over
